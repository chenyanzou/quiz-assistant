# 智能题库刷题助手 — 项目总体规划

## 一、项目概述

基于 **C++17** + **cpp-httplib** + **SQLite3** 的智能刷题系统。支持 JSON 格式题库导入、多模式刷题、AI 智能解析与举一反三出题、错题管理。**无需 Python**，整个项目编译为一个可执行文件 + 一个 HTML 文件。

## 二、技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 主后端 | C++17 + cpp-httplib | 高性能 HTTP 服务，RAII 风格 (header-only) |
| 数据库 | SQLite3 | 零配置本地数据库 (amalgamation) |
| JSON | nlohmann/json | header-only，现代 C++ JSON 库 |
| AI 调用 | libcurl | 多 AI 服务商 OpenAI 兼容 API |
| 前端 | Bootstrap 5 + Chart.js + 原生 JS | 单文件 SPA (`index.html`) |
| 构建 | g++ / MinGW / CMake | 跨平台 C++17 |

## 七、编译与运行

### 方式一：直接编译（推荐）

```bash
# 1. 编译 C 文件
gcc -c lib/sqlite3.c -o build/sqlite3.o -w

# 2. 编译 C++ 文件
g++ -std=c++17 -I. -Ilib -Ilib/curl -O2 -c database.cpp -o build/database.o
g++ -std=c++17 -I. -Ilib -Ilib/curl -O2 -c ai_service.cpp -o build/ai_service.o
g++ -std=c++17 -I. -Ilib -Ilib/curl -O2 -c quiz_server.cpp -o build/quiz_server.o

# 3. 链接
g++ -std=c++17 -O2 build/database.o build/ai_service.o build/quiz_server.o \
    build/sqlite3.o lib/libcurl.dll.a -lws2_32 -lcrypt32 -o quiz_server.exe
```

### 方式二：CMake（配置好 libcurl 开发库后）

```bash
mkdir build && cd build
cmake .. -G "MinGW Makefiles"  # 或 "Ninja"
cmake --build .
```

### 运行

```bash
# 默认端口 8080
./quiz_server.exe

# 自定义端口
./quiz_server.exe 3000

# 自定义端口和数据库路径
./quiz_server.exe 8080 data/my_quiz.db
```

### 浏览器访问

```
http://localhost:8080
```

## 八、功能模块

### 1. 题库导入模块
- 上传 JSON 文件导入题目
- 支持指定已有科目或自动创建新科目
- 显示导入结果统计

### 2. 题库管理模块
- 按科目/题型浏览所有题目
- 搜索、筛选、分页
- 查看题目详情（含 AI 解析）
- 手动添加题目（支持五种题型）
- 编辑、删除题目

### 3. 刷题模块
- **顺序刷题**：按题库顺序逐题作答
- **随机刷题**：随机抽取题目
- **模拟考试**：限定题目数，打乱顺序
- **错题重做**：只做错过的题目
- 即时反馈：作答后立即显示对错和解析
- 进度条显示

### 4. 错题本模块
- 自动收集错题（SQLite UPSERT 去重）
- 按科目分类查看
- 错题重做模式
- 标记已复习
- 统计图表

### 5. AI 智能模块（用户自配 API）
- 支持 OpenAI / Kimi / DeepSeek / 通义千问 / 智谱GLM / Ollama / 自定义 API
- **智能解析**：对已导入题目生成详细解析
- **举一反三**：根据题目生成同类变体题
- **AI 智能出题**：输入知识点自动生成题目 → 一键保存
- API Key 仅存浏览器 localStorage，不上传服务器

### 6. 学习统计模块
- 各科目正确率趋势（Chart.js 图表）
- 刷题数量统计
- 刷题历史记录

## 九、AI API 配置说明

用户打开网站后在「AI 助手 → AI 配置」页面自配 API，支持：

| 服务商 | Base URL | 推荐模型 |
|--------|----------|---------|
| OpenAI | `https://api.openai.com/v1` | gpt-4o-mini |
| Kimi | `https://api.moonshot.cn/v1` | moonshot-v1-8k |
| DeepSeek | `https://api.deepseek.com/v1` | deepseek-chat |
| 通义千问 | `https://dashscope.aliyuncs.com/compatible-mode/v1` | qwen-turbo |
| 智谱GLM | `https://open.bigmodel.cn/api/paas/v4` | glm-4-flash |
| Ollama | `http://localhost:11434/v1` | llama3 |
| 自定义 | 任意 OpenAI 兼容地址 | 按服务商文档 |

API Key 仅存浏览器 localStorage，不上传服务器。
